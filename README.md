# Calculatrice Python
Mini-projet de calculatrice en ligne de commande,
réalisé dans le cadre d'un cours sur Git et GitHub.
## Fonctionnalités
- Addition, soustraction, multiplication, division
- Gestion de la division par zéro
## Utilisation
```bash
python3 calcul.py
```
